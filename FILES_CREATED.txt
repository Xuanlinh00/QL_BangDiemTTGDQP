KEY INFORMATION
================================================================================

SYSTEM NAME: TVU GDQP-AN Admin Portal
PURPOSE: Admin-only system for managing 550+ PDF documents, running OCR, extracting student data, and generating reports

TECHNOLOGY STACK:
- Frontend: React 18 + TypeScript + Tailwind CSS
- Backend: Node.js (Express) + Python (FastAPI)
- Database: PostgreSQL 14+
- Cloud: AWS/Azure

TIMELINE:
- MVP: 8 weeks
- Phase 2: 4 weeks
- Total: 12 weeks

TEAM REQUIREMENTS:
- Frontend Developer: 1
- Backend Developer (Node.js): 1
- Backend Developer (Python): 1
- DevOps Engineer: 0.5
- QA Engineer: 0.5
- Total: 3.5 FTE

KEY FEATURES (MVP):
- Authentication (Week 1-2)
- Document Management (Week 3)
- OCR Processing (Week 4)
- Data Extraction (Week 5)
- Decision Management (Week 6)
- Reconciliation (Week 6)
- Dashboard & Reports (Week 7)
- Settings & Deployment (Week 8)

PERFORMANCE TARGETS:
- Dashboard load: < 2 seconds
- Document list load: < 3 seconds
- Scalability: 50,000+ students, 100,000+ scores

SECURITY:
- HTTPS only
- JWT authentication
- 2FA in Phase 2
- Audit logging
- Input validation
- SQL injection prevention
